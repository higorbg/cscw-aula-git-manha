Aula de Github - CSCW

Git é uma ferramenta de gerenciamento de versão, no qual vários membros de pertencentes a um mesmo projeto podem colaborar para a evolução do projeto. Basicamente os integrantes do mesmo projeto podem: importar os arquivos do diretório do projeto e adicionar ou substituir arquivos do presente projeto. O git soluciona um problema de comunicação entre pessoas de diferentes países e/ou turnos de trabalho entre vários integrantes de um mesmo projeto de forma a proporcionar um controle de modo organizado para todos os membros do projeto corrente. Além é claro de ajuda-los a "falarem a mesma lingua" na criação do projeto.
