Para que serve o GIT ? ? ?
O Git serve para o trabalho em grupo, aonde pessoas podem trabalhar em conjunto em um mesmo arquivo organizadamente. O CSCW tem o problema da distancia, aonde as pessoa precisam estar no mesmo local, o Git possibilita pessoas de diferentes partes do mundo a trabalharem no mesmo 'local virtual' sem nunca conhecerem pessoalmente umas as outras.
